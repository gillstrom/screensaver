# screensaver 

> Start the screensaver


## Install

```
$ npm install screensaver
```


## API

```js
const screensaver = require('screensaver');

screensaver().then(() => {
	console.log('Screensaver started');
});
```


## CLI

```
$ npm install --global screensaver
```

```
$ screensaver --help

  Usage
    $ screensaver
```
